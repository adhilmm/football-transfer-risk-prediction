# Football Player Transfer Risk Prediction

## Overview
A Machine Learning project that predicts the probability of a football player being transferred based on performance statistics, market value, age, and contract data.

## Tech Stack
- Python, Pandas, NumPy
- Scikit-learn, XGBoost, Random Forest
- Matplotlib, Seaborn
- Jupyter Notebook

## Project Workflow
1. Data Collection & Preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Model Building & Evaluation
5. Hyperparameter Tuning
6. Visualization of Results

## Real-World Impact
Helps football clubs and analysts predict transfer targets, optimize scouting decisions, and manage transfer budgets effectively.
